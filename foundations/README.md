# Foundations

This folder contains the foundational web development projects and exercises.

## Contents

- `index.html` - A starter HTML page demonstrating basic HTML structure
- `styles.css` - CSS styling to make the page look beautiful
- `script.js` - JavaScript for adding interactivity

## How to Use

1. Open `index.html` in your web browser to see the page in action
2. Try clicking the button to see JavaScript interactivity
3. Right-click on the page and select "Inspect" to see the HTML structure
4. Open the Console (F12) to see JavaScript console messages
