# Odin Project - Web Development Journey

This repository contains my learning progress and projects from [The Odin Project](https://www.theodinproject.com/), a free open-source coding curriculum for learning web development.

## About

The Odin Project provides a comprehensive curriculum for learning full-stack web development, covering:
- HTML & CSS
- JavaScript
- Git & GitHub
- Backend Development
- Frameworks and Libraries

## Repository Structure

```
/foundations       - HTML, CSS, and JavaScript basics
/projects         - Completed projects and exercises
/resources        - Helpful notes and reference materials
```

## Getting Started

Each folder contains projects and exercises completed as part of the curriculum. Feel free to explore and see my progress!
